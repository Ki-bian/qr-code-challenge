# QR Code Challenge

This project is a solution to the "QR Code Component" challenge from [Frontend Mentor](https://www.frontendmentor.io/challenges/qr-code-component-iux_sioz).

## Preview

![QR Code Component Preview](public/image-qr-code.png)

## Features

- Stylish QR code display
- Built with React and Tailwind CSS
- Responsive and modern design

## Installation

```sh
npm install
```

## Start the project

```sh
npm run dev
```

## Technologies used

- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)

## What I learned

This project allowed me to practice:

- Layout with Tailwind CSS
- Managing a modern React project
- Integrating a design from a mockup
